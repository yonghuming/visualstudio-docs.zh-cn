# <a name="ijsdebugbreakpointdelete-method"></a>IJsDebugBreakPoint::Delete 方法
删除断点。  
  
## <a name="syntax"></a>语法  
  
```  
HRESULT Delete(void);  
```  
  
## <a name="return-value"></a>返回值  
  
## <a name="remarks"></a>备注  
 如果在已删除断点上调用，则返回 S_FALSE。  
  
## <a name="requirements"></a>要求  
 **标头：** jscript9diag.h
