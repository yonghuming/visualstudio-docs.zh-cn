# <a name="nan-constant-javascript"></a>NaN 常量 (JavaScript)
一个特殊的常数，用于指定 thatNot 表达式不是数字。  
  
## <a name="syntax"></a>语法  
  
```  
NaN   
```  
  
## <a name="remarks"></a>备注  
 `NaN`常量 （而不是数字） 为属于`Global`对象，并可在初始化脚本引擎时。
