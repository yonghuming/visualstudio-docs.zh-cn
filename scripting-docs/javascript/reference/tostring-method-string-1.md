# <a name="tostring-method-string-1"></a>toString 方法 （字符串） 1
返回的字符串表示形式的字符串。  
  
## <a name="syntax"></a>语法  
  
```  
  
string.toString()  
```  
  
## <a name="parameters"></a>参数  
 `string`  
 必需。 要将其表示为一个字符串的数组。  
  
## <a name="return-value"></a>返回值  
 字符串的字符串表示形式。  
  
## <a name="example"></a>示例  
 下面的示例演示如何使用**toString**使用字符串的方法。  
  
```JavaScript  
var string = "this is a test";  
var strStr = string.toString();  
document.write(strStr);  
  
// Output: this is a test  
  
```
