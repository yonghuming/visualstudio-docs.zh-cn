# <a name="mathrandom-function-javascript"></a>Math.random 函数 (JavaScript)
返回介于 0 和 1 之间的伪随机数。  
  
## <a name="syntax"></a>语法  
  
```  
  
Math.random( )  
  
```  
  
## <a name="remarks"></a>备注  
 生成伪随机数是从 0 （含） 到 1 （独占），即，返回的值可以是零，但它将始终为小于 1。 当在自动种子的随机数生成器[!INCLUDE[javascript](../../javascript/includes/javascript-md.md)]首次加载。
