# <a name="mathtan-function-javascript"></a>Math.tan 函数 (JavaScript)
返回数字的正切值。  
  
## <a name="syntax"></a>语法  
  
```  
  
Math.tan(  
radians  
)  
  
```  
  
## <a name="remarks"></a>备注  
 所需`radians`自变量是一个包含以弧度计量的角度的数值表达式。  
  
 返回值的数值自变量的正切值`radians`。
