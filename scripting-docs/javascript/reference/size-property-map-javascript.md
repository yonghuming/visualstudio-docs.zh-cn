# <a name="size-property-map-javascript"></a>size 属性 (Map) (JavaScript)
返回映射中的元素数。  
  
## <a name="syntax"></a>语法  
  
```JavaScript  
sizeVar = mapObj.size  
```  
  
#### <a name="parameters"></a>参数  
 `sizeVar`  
 必需。 任意数值。  
  
 `mapObj`  
 必需。 任意 `Map` 对象。
