# <a name="resolvedoptions-method-intlcollator"></a>resolvedOptions 方法 (Intl.Collator)
返回一个包含属性和排序程序对象的值的对象。  
  
## <a name="syntax"></a>语法  
  
```  
collatorObj.resolvedOptions()  
```  
  
#### <a name="parameters"></a>参数  
 `collatorObj`  
 `Collator`要从中检索信息对象。  
  
## <a name="remarks"></a>备注  
 返回的对象的属性对应的计算属性`Collator`对象。
