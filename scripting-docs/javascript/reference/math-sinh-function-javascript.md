# <a name="mathsinh-function-javascript"></a>Math.sinh 函数 (JavaScript)
返回数字的反双曲正弦。  
  
## <a name="syntax"></a>语法  
  
```  
  
Math.sinh(x)   
```  
  
## <a name="remarks"></a>备注  
 返回值与 `(exp(x) - exp(-x))/2` 相同。 如果 x 为 `NaN`，则结果为 `NaN`。
