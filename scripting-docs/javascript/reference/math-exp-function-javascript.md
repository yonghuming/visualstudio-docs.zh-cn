# <a name="mathexp-function-javascript"></a>Math.exp 函数 (JavaScript)
返回 e （自然对数的底） 的幂。  
  
## <a name="syntax"></a>语法  
  
```  
  
Math.exp(number)   
```  
  
## <a name="remarks"></a>备注  
 所需`number`自变量是数值表达式表示的幂*e*。  
  
 返回值是一个数字。 常量*e*是欧拉数，约等于 2.71828 和`number`是提供的参数。  
  
 **适用于**: [Math 对象](../../javascript/reference/math-object-javascript.md)
