# <a name="valueof-method-map-javascript"></a>valueOf 方法 (Map) (JavaScript)
返回指定对象的基元值。  
  
## <a name="syntax"></a>语法  
  
```JavaScript  
mapObj.valueOf()  
```  
  
#### <a name="parameters"></a>参数  
 `mapObj`  
 必需。 一个 `Map` 对象。  
  
## <a name="property-valuereturn-value"></a>属性值/返回值  
 返回映射实例。
