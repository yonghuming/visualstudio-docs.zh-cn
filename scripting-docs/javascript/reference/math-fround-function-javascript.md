# <a name="mathfround-function-javascript"></a>Math.fround 函数 (JavaScript)
返回最近的单精度浮点格式的数字。  
  
## <a name="syntax"></a>语法  
  
```  
  
Math.fround(  
number  
)   
```  
  
## <a name="remarks"></a>备注  
 如果 `number` 为 `NaN`，则结果为 `NaN`。
