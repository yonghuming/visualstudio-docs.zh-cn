# <a name="division-assignment-operator---javascript"></a>除法赋值运算符 （-） (JavaScript)
将变量值除以表达式值，并将结果赋给该变量。  
  
## <a name="syntax"></a>语法  
  
```  
  
result /= expression   
```  
  
## <a name="parameters"></a>参数  
 `result`  
 任何数值变量。  
  
 `expression`  
 任何数值表达式。  
  
## <a name="remarks"></a>备注  
 使用`/=`运算符是与指定相同： `result = result / expression`。
