# <a name="constructor-property-weakset"></a>constructor 属性 (WeakSet)
指定创建 `WeakSet` 的函数。  
  
## <a name="syntax"></a>语法  
  
```JavaScript  
weakset.constructor  
```  
  
## <a name="remarks"></a>备注  
 所需的 `weakset` 是集合的名称。  
  
 `constructor` 属性是具有原型的每个对象的原型的成员。 它包括所有内部 JavaScript 对象，`Global` 和 `Math` 对象除外。 `constructor` 属性包含对构造特定对象实例的函数的引用。
