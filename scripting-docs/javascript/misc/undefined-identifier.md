# <a name="undefined-identifier"></a>未定义标识符
你尝试使用标识符的[!INCLUDE[javascript](../../javascript/includes/javascript-md.md)]编译器无法识别。 每当你使用，将返回未定义的值：  
  
-   不存在，变量  
  
-   已声明，但从未过的变量分配给它的值  
  
-   对象属性不存在。  
  
### <a name="to-correct-this-error"></a>更正此错误  
  
-   声明的变量进行**var**语句 (如`var`x;)。
