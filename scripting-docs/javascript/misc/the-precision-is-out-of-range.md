# <a name="the-precision-is-out-of-range"></a>精度超出范围
尝试向函数传递了无效自变量**Number.prototype.toPrecision**。 自变量**toPrecision**必须介于 1 和 21 （含） 之间。  
  
### <a name="to-correct-this-error"></a>更正此错误  
  
-   确保自变量`toPrecision`不太大或太小。
