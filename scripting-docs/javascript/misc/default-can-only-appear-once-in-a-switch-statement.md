# <a name="39default39-can-only-appear-once-in-a-39switch39-statement"></a>&#39; 默认 &#39;只能出现一次在 &#39; 交换机 &#39;语句
你尝试使用**默认**多次 switch 语句中的语句。 默认情况下始终是 switch 语句 （它是秋季通过如此） 中的最后一个 case 语句。  
  
### <a name="to-correct-this-error"></a>更正此错误  
  
-   删除任何额外**默认**case 语句从你`switch`语句 （使用在 switch 语句中的大多数一个默认 case 语句）。
