# <a name="expected-3939-javascript"></a>预期 &#39; = &#39;(JavaScript)
你试图创建用于条件编译语句的变量，但未将放置到的变量与你想要向其分配的值之间用等号。  
  
### <a name="to-correct-this-error"></a>更正此错误  
  
-   添加一个等号。 例如:   
  
    ```JavaScript  
    @set @myvar1 = 1  
    ```
