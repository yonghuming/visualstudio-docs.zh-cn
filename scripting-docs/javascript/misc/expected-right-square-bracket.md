# <a name="expected-3939"></a>预期 &#39;]&#39;
你进行对数组元素的引用，但不是包括右括号。 数组元素是指任何表达式必须包括同时打开和右括号。  
  
### <a name="to-correct-this-error"></a>更正此错误  
  
-   添加为引用的数组元素的表达式的右括号。
