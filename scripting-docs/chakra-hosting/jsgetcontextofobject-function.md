# JsGetContextOfObject 函数
获取对象所属的脚本上下文。  
  
## 语法  
  
```  
STDAPI_(JsErrorCode) JsGetContextOfObject(  
  _In_ JsValueRef object,  
  _Out_ JsContextRef *context  
);  
```  
  
#### 参数  
 `object`  
 要从其获取上下文的对象。  
  
 `context`  
 对象所属的上下文。  
  
## 返回值  
 如果该操作成功，则为代码 `JsNoError`，否则为失败代码。  
  
## 备注  
 需要活动脚本上下文。  
  
## 要求  
 **标头：**jsrt.h
