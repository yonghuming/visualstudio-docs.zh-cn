# 调试准备：C#、F# 和 Visual Basic 项目类型
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

本节主题描述如何调试由 [!INCLUDE[vsprvs](../code-quality/includes/vsprvs_md.md)] 项目模板创建的 C\#、F\# 和 [!INCLUDE[vbprvb](../code-quality/includes/vbprvb_md.md)] 项目类型。  
  
 注意，那些将 DLL 作为其输出创建的项目类型由于其共同的特性已被分组到[调试 DLL 项目](../debugger/debugging-dll-projects.md)。  
  
## 本节内容  
 [推荐的属性设置](../debugger/managed-debugging-recommended-property-settings.md)  
 本节描述 C\#、F\# 和 Visual Basic 项目中推荐的与调试有关的属性设置。  
  
 [Windows 窗体应用程序](../debugger/debugging-preparation-windows-forms-applications.md)  
 描述 Windows 应用程序项目，并提供有关调试、更改默认调试配置以及从 [!INCLUDE[vsprvs](../code-quality/includes/vsprvs_md.md)] 外部启动应用程序并附加到它上面的说明。  
  
 [控制台项目](../debugger/debugging-preparation-console-projects.md)  
 提供调试 C\# 或 [!INCLUDE[vbprvb](../code-quality/includes/vbprvb_md.md)] 控制台应用程序的附加注意事项。  这包括：指定命令行参数、从命令提示处启动应用程序、写入“输出”窗口以及“控制台”窗口的疑难解答。  
  
 [Windows 服务](../debugger/debugging-preparation-windows-services.md)  
 描述 Windows 服务并提供指向“调试 Windows 服务应用程序”的链接。
