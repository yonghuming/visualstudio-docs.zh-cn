# <a name="message-properties-dialog-box"></a>“消息属性”对话框
使用此对话框中，若要了解有关特定消息的详细信息。 若要显示此对话框中，将焦点移至[消息视图](../debugger/messages-view.md)窗口。 在树中，选择任何消息节点，然后选择**属性**从**视图**菜单。  
  
 **常规**选项卡是显示的唯一选项卡。 以下设置将可用：  
  
 **窗口句柄**  
 此窗口的唯一 ID。 窗口句柄数字会重复使用;它们仅针对该窗口的生存期标识一个窗口。 单击此值可查看此窗口的属性。  
  
 **嵌套级别**  
 此消息，其中 0 表示没有嵌套的嵌套深度。  
  
 **消息**  
 数、 状态和选择的 windows 消息的名称。  
  
 **lresult，这在**  
 值*lResult*参数，如果有的话。  
  
 **wParam**  
 值*wParam*参数，如果有的话。  
  
 **lParam**  
 值*lParam*参数，如果有的话。 此值进行解码后，如果它是指向字符串或结构的指针。
