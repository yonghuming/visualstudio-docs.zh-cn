# <a name="window-properties-dialog-box"></a>“窗口属性”对话框
使用此对话框中，若要了解有关特定窗口的详细信息。 若要显示此对话框中，将焦点移至[Windows 视图](../debugger/windows-view.md)窗口。 在树中，选择任何窗口节点，然后选择**属性**从**视图**菜单。  
  
## <a name="in-this-section"></a>本节内容  
 [常规选项卡。窗口属性对话框](../debugger/general-tab-window-properties-dialog-box.md)  
 显示有关所选的窗口的信息。  
  
 [样式选项卡。窗口属性对话框](../debugger/styles-tab-window-properties-dialog-box.md)  
 显示应用于所选的窗口样式。  
  
 [Windows 选项卡。窗口属性对话框](../debugger/windows-tab-window-properties-dialog-box.md)  
 在与所选的窗口相关的 windows 上显示信息。  
  
 [类选项卡。窗口属性对话框](../debugger/class-tab-window-properties-dialog-box.md)  
 显示有关所选的窗口类信息。  
  
 [进程选项卡。窗口属性对话框](../debugger/process-tab-window-properties-dialog-box.md)  
 标识所选的窗口的过程。
