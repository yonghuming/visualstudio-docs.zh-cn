# <a name="help-menu-commands"></a>“帮助”菜单命令
**帮助**菜单提供用于打开帮助文件的命令。  
  
 **索引**  
 打开 Spy + + 帮助内容屏幕。  
  
 **使用帮助**  
 将打开介绍如何使用帮助的帮助文件。  
  
 **关于 Spy + +**  
 显示 Spy + + 中的软件版本和版权信息。
