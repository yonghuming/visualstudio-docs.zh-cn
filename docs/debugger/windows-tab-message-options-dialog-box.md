# <a name="windows-tab-message-options-dialog-box"></a>“消息选项”对话框 ->“窗口”选项卡
使用**Windows**选项卡以设置消息中列出的范围[消息视图](../debugger/messages-view.md)。 若要显示[消息选项对话框](../debugger/message-options-dialog-box.md)，选择**日志消息**从**Spy**菜单。  
  
 以下设置子网上有**Windows**选项卡：  
  
 **查找程序工具**  
 用于选择一个窗口的拖放工具。  
  
 **隐藏 Spy + +**  
 移除 Spy + + 主窗口，但仍**消息**对话框中显示在其他应用程序的前面。  
  
 **父窗口**  
 显示有关所选的窗口和其直接父窗口的消息。  
  
 **子窗口**  
 显示所选的窗口和所有子窗口，包括嵌套的子窗口的消息。  
  
 **同一线程的窗口**  
 显示有关所选的窗口和相同的线程所拥有的所有其他 windows 消息。  
  
 **同一进程的窗口**  
 显示有关所选的窗口和相同进程所拥有的所有其他 windows 消息。  
  
 **在系统中的所有窗口**  
 显示所有 windows 消息。  
  
 **将设置保存为默认值**  
 保存新消息流窗口的前面设置。 退出 Spy + + 时，还会保存这些设置。
