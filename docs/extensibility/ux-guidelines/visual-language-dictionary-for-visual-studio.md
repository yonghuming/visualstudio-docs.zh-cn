# <a name="visual-language-dictionary-for-visual-studio"></a>Visual Studio 的 visual 语言字典
Visual Studio 使用公共 visual 语言演示是其插图和其他图像的构建基块的概念。 在设计新图标，使用此字典来标识你要在你自己的映像中传达的概念。  
  
## <a name="general-guidelines"></a>通用准则
  
-   这些 visual 概念必须一致地使用其字典定义。 否则，则可能令人费解你预期要具有一致的含义的常见元素的用户。  
  
-   以便图像清晰且易于区分，请在您的图标，选择仅使用的最相关或重要概念。  
  
-   在每个图像中使用只能将一个或两个概念。  
  
-   设计图标的多个策略，请参阅[图标设计](../../extensibility/ux-guidelines/images-and-icons-for-visual-studio.md#BKMK_IconDesign)。
