# 使用 Visual C# 开发环境
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Visual C\# 集成开发环境 \(IDE\) 是一种通过常用用户界面公开的开发工具的集合。  有些工具是与其他 [!INCLUDE[vsprvs](../code-quality/includes/vsprvs_md.md)] 语言共享的，还有一些工具（如 C\# 编译器）是 Visual C\# 特有的。  本主题提供指向最重要的 Visual C\# 工具的链接。
