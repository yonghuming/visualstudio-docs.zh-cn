# 仅我的代码（线程视图）
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

如果选择此选项，将筛选调用堆栈，以仅显示代码和一级被调用的函数。  
  
 通过激活此选项，可以极大降低调用堆栈的复杂性，还可以简化对特定问题的诊断。  
  
 在某些情况下，选择此选项可以筛选出阻止的调用。  如果需要完整的调用堆栈详细信息以进行决定，请清除此选项以公开完整的调用堆栈。
