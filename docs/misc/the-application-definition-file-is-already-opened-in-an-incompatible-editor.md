# 应用程序定义文件已在不兼容的编辑器中打开
**应用程序定义文件已在不兼容的编辑器中打开。  请关闭其他编辑器并重新加载项目属性页。**  
  
 尝试打开应用程序定义文件并且该文件已经在另一个编辑器中打开时，会发生此错误。  
  
### 更正此错误  
  
1.  关闭其他编辑器。  
  
2.  关闭“项目设计器”。  
  
3.  在**“项目”**菜单上单击**“属性”**，重新打开“项目设计器”。
