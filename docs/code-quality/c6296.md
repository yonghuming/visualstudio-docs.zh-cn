# <a name="c6296"></a>C6296
警告 C6296: for 循环： 循环正文只能执行一次  
  
 此警告意味着 for 循环可能不会按预期。 索引无符号后，向下，从零计数循环，其正文是只运行一次。  
  
## <a name="example"></a>示例  
 下面的代码将生成此警告：  
  
```  
void f( )  
{  
   unsigned int i;  
  
   for (i = 0; i < 100; i--)  
   {  
      // code ...  
   }  
}  
```  
  
 若要更正此警告，请使用下面的代码：  
  
```  
void f( )  
{  
   unsigned int i;  
  
   for (i = 0; i < 100; i++)  
   {  
      // code ...  
   }  
}  
```
