# C28239
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

警告 C28239：注解的参数同时具有 value、stringValue 或 longValue； paramn\=xxx。  仅使用 paramn\=xxx  
  
 PREfast 模型文件发生了内部错误。  此警告在通常应用时不应当发生。
