# <a name="c6385"></a>C6385
警告 C6385： 无效的数据： 访问\<缓冲区名称 >，可读的大小是\<size1 > 个字节，但\<size2 > 可能读取的字节数： 行： x，y  
  
 此警告指示指定的缓冲区的可读范围可能小于用于从中读取信息的索引。 尝试读取超出有效范围的数据会导致缓冲区溢出。  
  
## <a name="example"></a>示例  
 下面的代码将生成此警告：  
  
```  
void f(int i)  
{  
   char a[20];  
   char j;  
   if (i <= 20)  
   {  
      j = a[i];  
   }  
}  
```  
  
 若要更正此警告，请使用下面的代码：  
  
```  
void f(int i)  
{  
   char a[20];  
   char j;  
   if (i < 20)  
   {  
      j = a[i];  
   }  
}   
```
