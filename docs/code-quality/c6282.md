# <a name="c6282"></a>C6282
警告 C6282： 运算符不正确： 在布尔上下文中的常数的赋值。 请考虑使用 = = 改为  
  
 此警告意味着在测试上下文中检测到的常量为变量赋值。 分配给测试上下文中的变量几乎始终是常量的不正确的。 替换`=`与`==`，或从要解决此警告的测试上下文中删除的分配。  
  
## <a name="example"></a>示例  
 下面的代码将生成此警告：  
  
```  
void f( int i )  
{  
   while (i = 5)  
   {  
   // code    
   }  
}  
```  
  
 若要更正此警告，请使用下面的代码：  
  
```  
void f( int i )  
{  
   while (i == 5)  
   {  
   // code    
   }  
}  
```
