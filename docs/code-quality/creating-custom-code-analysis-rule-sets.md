# <a name="creating-custom-code-analysis-rule-sets"></a>创建自定义代码分析规则集
通过指定包含一个或多个规则的规则集可将代码分析规则应用于托管的代码项目中。 Microsoft 标准规则集提供的按函数和深度组织的规则的组。 例如，Microsoft 基本设计准则规则和 Microsoft 扩展设计准则规则包含专注于可用性和可维护性问题，与添加主要侧重于在扩展规则集中的命名规则的规则。  
  
 你还可以创建自定义规则集组合了两个或多个标准规则集，或指定一个或多个标准规则集中的规则。 保存自定义规则设置为的.ruleset 文件，并可以在多个代码项目中指定它们。
