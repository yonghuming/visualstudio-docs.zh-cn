# <a name="c28224"></a>C28224
警告 C28224： 批注需要参数  
  
 此警告意味着所命名的批注使用不带参数的并在至少一个参数是必需的。 如果对批注使用标准宏，则这种情况应该是不可能的；正常使用时不应出现此警告。
