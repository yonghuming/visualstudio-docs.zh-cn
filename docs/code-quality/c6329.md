# C6329
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

警告 C6329:不应针对 \<数字\>进行调用\<函数\>的返回值检查  
  
 程序比较数字和调用`CreateFile`的返回值。  `CreateFile` 如果成功，它将返回一个开启句柄到对象。  如果失败，则返回 `INVALID_HANDLE_VALUE`：  
  
## 示例  
 此代码会导致警告：  
  
```cpp  
if (CreateFile() == NULL)  
    {  
        return;  
    }  
```  
  
## 示例  
 代码更正此错误：  
  
```cpp  
if (CreateFile() == INVALID_HANDLE_VALUE)  
    {  
        return;  
    }  
```
